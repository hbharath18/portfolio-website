# 🧑‍💻 Portfolio Website

A personal portfolio website built with HTML5, CSS3, and Bootstrap 4 — showcasing experience, education, certifications, and skills.

🔗 **Live Demo:** [portfolio-website-zeta-gules-81.vercel.app](https://portfolio-website-zeta-gules-81.vercel.app)

---

## 📸 Preview

> A clean, responsive portfolio with a sticky navbar, card-style sections, and smooth hover animations.

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3, Bootstrap 4 |
| Fonts | Poppins (Google Fonts) |
| Deployment | Vercel |

---

## 📁 Project Structure

```
portfolio-website/
├── index.html       # Main HTML file
├── pstyle.css       # Custom CSS styles
└── Photo.png        # Profile photo
```

---

## ✨ Features

- Responsive layout using Bootstrap 4 grid system
- Sticky navbar with smooth scroll navigation
- Hover animations on section cards
- Sections: Home, Experience, Education, Certifications, Skills
- Deployed on Vercel with GitHub CI/CD integration

---

## 🚀 Getting Started

### Run Locally

Just open the HTML file directly in your browser — no build step needed:

```bash
# Clone the repo
git clone https://github.com/hbharath18/portfolio-website.git

# Open in browser
open portfolio.html
```

### Deploy on Vercel

Connected to GitHub — any push to the `main` branch triggers an automatic redeployment.
